# InowDocs

Selamat datang di **InowDocs** — pusat dokumentasi produksi tim Inowtech.

Repo ini berisi panduan praktis untuk tim produksi: dari mapping stack,
panduan setup, workflow & SOP kerja, hingga catatan troubleshooting.

## Daftar Isi

- **Mapping Stack** — gambaran besar teknologi yang dipakai (Laravel, Cloudflare, database, dll).
- **Panduan Setup** — langkah instalasi & konfigurasi (install Laravel, setup Cloudflare tunnel).
- **Workflow & SOP** — alur kerja tim (git flow, testing flow).
- **Troubleshooting** — error umum & solusinya.

## Cara Kontribusi

1. Edit / tambah file markdown di folder `docs/`.
2. Jalankan `mkdocs serve` untuk preview lokal.
3. Push ke branch `main` → Cloudflare Pages otomatis deploy.

!!! tip "Catatan"
    Setiap halaman yang selesai dibuat dan diverifikasi bisa dicatat ke KPI
    dokumentasi (m1–m4) di tracker Kang Ahmad.
